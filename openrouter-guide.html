Using OpenRouter.ai with the Draconic Translator

OpenRouter.ai provides access to a wide variety of Large Language Models (LLMs), including some free, rate-limited models. This guide will help you set it up for use with the Draconic Translator.

Steps:

  1. Sign Up/Log In to OpenRouter: Go to https://openrouter.ai and create an account or log in.
  2. Get Your API Key:
    • Once logged in, navigate to your account settings (usually accessible via your profile icon on the top right) and select "Keys" or look for an "API Keys" section.
    • Click "Create Key". Give it a descriptive name (e.g., "DraconicTranslator").
    • Copy the generated API key. Keep this key secure, like a password. OpenRouter will only show it to you once.
  3. Configure Draconic Translator Settings:
    • Open the Draconic Translator and go to the Settings page.
    • API Key: Paste the API key you copied from OpenRouter into the "API Key" field.
    • Base URL: Change the "Base URL" field to: https://openrouter.ai/api/v1
    • Select Model:
      • OpenRouter supports many models. Some free models you might try include (check OpenRouter for current availability and identifiers):
        • deepseek/deepseek-r1:free
        • deepseek/deepseek-chat-v3-0324:free
        • deepseek/deepseek-r1-0528:free
        • Other models listed with a $0 price on OpenRouter, especially other Deepseek models if available.
        • meta-llama/llama-4-maverick:free for image support
      • To use these, select "Use Custom Model Name â†“" from the dropdown in the translator settings, and then enter the full model identifier (e.g., deepseek/deepseek-r1:free) in the "Custom Model Name" field.
      • Always check the OpenRouter Models page for the latest list of available models, their identifiers, and pricing.
  4. Save and Test:
    • In the Draconic Translator settings, click "Save Settings".
    • Then, click "Test Connection" to ensure everything is configured correctly. You should see a "Connection successful!" message.

Important Notes:

  • Free Models & Rate Limits: Free models on OpenRouter are typically rate-limited. For many free tier models, this is often around 50 Requests Per Day (RPD). If you encounter errors or slow responses, you might have exceeded these limits.
  • Model Performance: The quality of translations can vary significantly between models. Experiment to find one that works well for Draconic.
    • Smaller or free models might not be as capable as larger, paid models.
    • Highly capable paid models include options like anthropic/claude-3.7-sonnet or google/gemini-2.5-pro.
    • However, some free or very low-cost models like deepseek/deepseek-r1:free can offer performance that compares favorably to these premium models for specific tasks, so it's worth testing.
  • Costs: While some models are free (like the Deepseek ones mentioned), OpenRouter also provides access to many paid models. Be aware of the pricing if you choose to use models like Claude or Gemini Pro. You typically need to add credits to your OpenRouter account for paid models.
  • Referral Header: For some models, especially free ones, OpenRouter might require or prefer you to set an HTTP Referer header. This translator automatically sets X-Title to "Draconic Translator" in requests, which OpenRouter encourages. You can also set your site URL in OpenRouter's key settings if you deploy this application.

You should now be able to use the Draconic Translator with models hosted on OpenRouter.ai!
