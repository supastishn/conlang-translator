Using Google AI Studio for Gemini Models with Draconic Translator

Google AI Studio allows you to get API keys for Gemini models, which can be used with services offering OpenAI-compatible endpoints. This guide shows how to set up the Draconic Translator to use Gemini models via Google's own OpenAI-compatible proxy.

Steps:

  1. Go to Google AI Studio: Visit https://aistudio.google.com/ and sign in with your Google account.
  2. Get Your API Key:
    • Once logged in, click on "Get API key" (often found in the top left or via a navigation menu).
    • Follow the instructions to create a new API key. You might need to create a new project if you don't have one already.
    • Copy the generated API key. Keep this key secure.
  3. Configure Draconic Translator Settings:
    • Open the Draconic Translator and go to the Settings page.
    • API Key: Paste the API key you copied from Google AI Studio into the "API Key" field.
    • Base URL: Change the "Base URL" field to Google's OpenAI-compatible endpoint: https://generativelanguage.googleapis.com/v1beta/openai/
    • Select Model:
      • To use a Gemini model, select "Use Custom Model Name â†“" from the model dropdown.
      • In the "Custom Model Name" field that appears, enter the specific Gemini model identifier. Recommended free and capable models include:
        • gemini-2.5-flash-preview-04-17
        • gemini/gemini-2.5-flash-preview-05-20 (Note: some Gemini models require the gemini/ prefix for the OpenAI-compatible endpoint)
      • You can find other available Gemini models by checking Google's official documentation. The model identifier might need to be prefixed (e.g., gemini/model-name) or used as-is depending on the exact proxy requirements. Start with the formats shown.
  4. Save and Test:
    • In the Draconic Translator settings, click "Save Settings".
    • Then, click "Test Connection" to ensure everything is configured correctly. You should see a "Connection successful!" message.

Important Notes:

  • Recommended Models & Rate Limits: Models like gemini-2.5-flash-preview-04-17 or gemini/gemini-2.5-flash-preview-05-20 (or similar "flash" versions) are often available with a generous free tier, potentially around 500 Requests Per Day (RPD) via Google AI Studio. Always check Google's current terms and rate limits.
  • OpenAI Compatibility: The Base URL https://generativelanguage.googleapis.com/v1beta/openai/ is a proxy provided by Google that makes their Gemini API compatible with OpenAI API standards.
  • Model Identifiers: Ensure you use the correct model identifier for the OpenAI-compatible endpoint. This might sometimes differ slightly from identifiers used in native Gemini SDKs. The gemini/model-name format is a common convention for proxied services, but some models might be accessible without the prefix. Test with both if unsure.
  • Terms of Service: Be sure to review Google AI Studio's and Gemini's terms of service.

By following these steps, you can leverage Google Gemini models for your translations directly through Google's OpenAI-compatible API.
