API Configuration

How to use:

  1. Enter your OpenAI API key or any compatible API key
  2. Set the API URL for your chosen provider
  3. Select the model you want to use for translation
  4. Click "Save Settings" to store your preferences
  5. Return to the translator page to start using the app

API settings are stored locally in your browser
